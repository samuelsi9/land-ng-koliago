# Legacy static Koliago site

This folder is the **original** landing page built with **HTML, CSS, and JavaScript** (no build step).

## Files

- **`index.html`** — main page (previously `www.html` at repo root)
- **`css/`** — stylesheets (`general.css`, `style.css`, `queries.css`)
- **`javascript/script.js`** — navigation, modal, smooth scroll, etc.
- **`img/`**, **`downloads/`**, **`manifest.webmanifest`** — assets

## How to open locally

- Double‑click **`index.html`**, or  
- From this directory run a static server, e.g. `npx serve .` or `python -m http.server 8080`

## Note

The **deployable** app for GitHub / production is the **React + Vite** project at the **repository root** (`npm run dev` / `npm run build`). This folder is kept for reference and comparison only.
